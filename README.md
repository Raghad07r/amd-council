# مجلس مستشار أمد للوعي المالي
## هاكاثون أمد 2026 — FastAPI + OpenRouter Edition

نظام متعدد الوكلاء (Multi-Agent System) يضم ثلاثة مستشارين ماليين يتفاعلون بشكل ديناميكي حسب طبيعة السؤال.

---

## المستشارون

| الاسم | الدور | يتدخل عند |
|-------|-------|-----------|
| **سلمان** | المخطط المالي | الميزانية، الادخار، الخطط المستقبلية |
| **نورة** | محللة المخاطر | الاستثمار، الأسهم، العقار، تقييم المخاطر |
| **فهد** | خبير السلوك | العادات الشرائية، الإسراف، الجانب النفسي |

---

## المتطلبات

- Python 3.11+
- مفتاح API من [openrouter.ai](https://openrouter.ai/keys)

---

## التثبيت والتشغيل

```bash
# 1. استنسخ المشروع
git clone ...
cd amd-council

# 2. ثبّت المكتبات
pip install -r requirements.txt

# 3. اضبط المتغيرات البيئية
cp .env.example .env
# افتح .env وأضف OPENROUTER_API_KEY

# 4. شغّل الخادم
python server.py
# أو
uvicorn server:app --reload --port 3000
```

ثم افتح: http://localhost:3000

## آلية العمل الديناميكية (Dynamic Routing)

1. **التقييم المسبق**: قبل أن يرد أي مستشار، يجري النظام تقييماً سريعاً عبر مطابقة الكلمات المفتاحية، وعند الغموض يستشير النموذج مباشرة.
2. **الرد المتسلسل الانتقائي**: يمر على المستشارين الثلاثة بالتوالي، ويتخطى من لا يعنيه السؤال.
3. **بث فوري**: كل توكن يصل مباشرة للمتصفح عبر SSE بدون انتظار.

## آلية الطابور (Queue Mechanism)

- **حقل الإدخال لا يُقفل أبداً** أثناء حديث المجلس.
- إذا أرسل المستخدم رسالة والمجلس مشغول، تظهر كـ **رسالة معلقة** (Pending).
- عند انتهاء الدورة الحالية، يُسحب الطلب القادم **تلقائياً** من الطابور.
